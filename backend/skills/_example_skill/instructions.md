# Example Skill Instructions

This is an example skill for testing Nova's dynamic skills system.

## Purpose

This skill demonstrates how skills work in Nova. It provides simple greeting tools that can be used to verify the skill activation and deactivation flow.

## Available Tools

When this skill is activated, you gain access to the following tools:

1. **say_hello** - Generate a friendly greeting for a given name
2. **say_goodbye** - Generate a farewell message for a given name

## Usage Guidelines

- Use `say_hello` when the user wants to greet someone
- Use `say_goodbye` when the user wants to bid farewell to someone
- These are simple demonstration tools and should be used for testing purposes

## Example Workflow

1. User asks to greet someone
2. Use the `say_hello` tool with the person's name
3. Return the greeting to the user
